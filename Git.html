🐱Gitの使い方！

Gitとは、それぞれの作ったプログラムソースなどの変更履歴を管理することができるシステムです！

特徴としては、ローカル環境(自分のPC内)にもコードの変更履歴を保存することができ、
個々のローカル環境がリポジトリのサーバーになることが可能になります！
それにより、ネットワークに接続していなくても作業をすることができます！

💻コマンド一覧表💻

コンテンツ 説明
ssh -T git@github.com Githubと連結ができているかの確認コード
git init リポジトリを作成する。
git status Gitの状態を表示させる。
git add . ファイルをステージングする
git commit -m ~ ファイルをコミットする
git remote add origin ~ Githubと同期させる
git push -u origin master ファイルをGithubにアップロードさせる
