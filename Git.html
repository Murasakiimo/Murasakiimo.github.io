🐱Githubの使い方

リポジトリ作成

アカウントに登録後、"New Repository"を選択します。
Repository name"の部分に好きな名前でリポジトリを作成します。
"Description"の部分にコメントを残します。

リポジトリの中にあらかじめREADMEファイルを作成しておく場合は、
「Initialize this repository with a README」にチェックします！
リモートリポジトリのアドレスはアップロード時に使うので、控えておいてください！

ファイルを作成・編集してアップロード！

ローカルのPCにリポジトリを作成し、本ページ下記の"コマンド一覧表"を見ながらアップロードしていきます！

🐱Github-pagesの作り方

Repository nameに[ GitHubアカウント名 ].github.io と入力します！
緑の"Create repository"をクリック

ローカル作業

ターミナルを起動し、下記を実行して下さい。
"git clone https://github.com/ [GitHubアカウント名]/[GitHubアカウント名].github.io"

実行することでローカルにファイルが生成されています。
ファイル名：[GitHubアカウント名].github.io

実行の指定先を移動します。
cd [GitHubアカウント名].github.io


Htmlファイルを作り、GithubへPushしたら

完了です！

💻コマンド一覧表💻

コンテンツ 説明
git init 初期化、Gitリポジトリを新たに作成するコマンドです！
git add ファイル名 インデックス領域にあげる
git add . インデックス領域に変更ファイルすべてをあげる
git commit インデックス領域にあるファイルをコミット
git commit -m ”コメント” Commitのコメント簡易版
git status Gitの状態を表示させる。
git log gitのログを見る
git log –pretty short logの簡易表示版
git diff 変更箇所がわかる
git push Git Hubへあげる
git pull Git Hubから落としてくる
git branch ブランチ名 ブランチの作成
git checkout ブランチ名 ブランチの移動
git push -u origin Git Hubへブランチをあげる
git branch -d ブランチ名 ブランチの削除
git branch 現状のブランチを見る
git merge –no-ff ブランチ名 ブランチのマージ
git log –graph ブランチの流れを可視化
